## Разработка

- Установка зависимостей: `npm install`
- Старт сервера для локальной разработки: `npm start`
- Запуск сборки тестов: `npm run test`
- Запуск сборки для деплоя: `npm run build`
- Сборка и деплой: `npm run deploy`
